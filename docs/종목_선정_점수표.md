# 종목 선정 점수표

총 **100점** 기준의 종목 선정 평가 체계. 정량 지표 70점과 정성 지표 30점으로 구성된다.

---

## 1. 지표 분류 요약

| 구분 | 카테고리 | 배점 | 비중 |
|---|---|---|---|
| 정량 | 이익 창출력 / 저평가 | 30점 | 30% |
| 정량 | 주주환원 의지 | 40점 | 40% |
| 정성 | 이익 지속성 | 5점 | 5% |
| 정성 | 미래 성장 잠재력 / 기업 경쟁력 | 25점 | 25% |
| **합계** | | **100점** | **100%** |

- 정량 : 정성 = **70 : 30**
- 주주환원(40점)이 가장 큰 비중을 차지하며, 배당과 자사주 정책을 중시
- 저평가(PER + PBR) 25점으로 가치투자 성향 반영

---

## 2. 정량 지표 (70점)

### 2.1 이익 창출력 / 저평가 (30점

| 지표 | 구간 및 점수 | 만점 |
|---|---|---|
| **PER** | `<5` → 20점 / `<8` → 15점 / `<10` → 10점 / `>10` → 5점 | 20 |
| **PBR** | `<0.3` → 5점 / `<0.6` → 4점 / `<1.0` → 3점 / `>1.0` → 0점 | 5 |
| **중복 상장 여부** (자회사·손자회사 상장) | 단독 상장 → 5점 / 중복 상장 → 0점 | 5 |

### 2.2 주주환원 의지 (40점)

| 지표 | 구간 및 점수 | 만점 |
|---|---|---|
| **배당수익률** | `>7%` → 10점 / `>5%` → 7점 / `>3%` → 5점 / `<3%` → 2점 | 10 |
| **분기 배당 실시** | 예 → 5점 / 아니오 → 0점 | 5 |
| **배당 연속 인상 연수** (동결 시 연속 인정하나 인상 불인정) | 10년 이상 → 5점 / 5년 이상 → 4점 / 3년 이상 → 3점 / 해당 없음 → 0점 | 5 |
| **정기적 자사주 매입·소각** (최소 연 1회 이상) | 예 → 7점 / 아니오 → 0점 | 7 |
| **연간 소각 비율** (총주식수 대비) | `>2%` → 8점 / `>1.5%` → 5점 / `>0.5%` → 3점 / `<0.5%` → 0점 | 8 |
| **자사주 보유 비율** | 없음 → 5점 / `<2%` → 4점 / `<5%` → 2점 / `>5%` → 0점 | 5 |

---

## 3. 정성 지표 (30점)

### 3.1 이익 지속성 (5점)

| 지표 | 구간 및 점수 | 만점 |
|---|---|---|
| **이익 지속 가능성** | 대체로 지속 가능 → 5점 / 불안정한 이익 창출력 → 0점 | 5 |

### 3.2 미래 성장 잠재력 / 기업 경쟁력 (25점)

| 지표 | 구간 및 점수 | 만점 |
|---|---|---|
| **미래 성장 잠재력** | 매우 높다 → 10점 / 높다 → 7점 / 보통 → 5점 / 낮다 → 3점 | 10 |
| **기업 경영** | 우수한 경영자 → 10점 / 전문 경영자 → 5점 / 저조한 실적 오너 경영 → 0점 | 10 |
| **세계적 브랜드 보유 여부** | 있다 → 5점 / 없다 → 0점 | 5 |

---

## 4. 배점 인사이트

- **최고 배점 TOP**: PER(20) > 배당수익률 · 미래 성장 잠재력 · 기업 경영 (각 10)
- **주주환원 40점**: 배당 + 자사주 정책 중시 (배당 25점 + 자사주 20점)
- **저평가 지표 25점**: PER + PBR → 가치투자 관점
- **정성 평가 30점**: 숫자만으로 판단하지 않는 균형 구조

---

## 5. 지표별 데이터 소스

| 지표 | 데이터 소스 |
|---|---|
| PER | pykrx (`get_market_fundamental`) |
| PBR | pykrx (`get_market_fundamental`) |
| 이익 지속 가능성 | KIS `finance_income_statement` / DART 재무제표 (연·분기 손익) |
| 중복 상장 여부 | DART 종속회사 현황 + KRX 상장 종목 매칭 |
| 배당수익률 | pykrx (`get_market_fundamental` DIV) |
| 분기 배당 실시 | KIS `ksdinfo_dividend` → `stock_dividends` 테이블 (`DividendService.is_quarterly_dividend`) |
| 배당 연속 인상 연수 | KIS `ksdinfo_dividend` → `stock_dividends` 테이블 (`DividendService.consecutive_dividend_increase_years`) |
| 정기적 자사주 매입·소각 | DART 공시 (자기주식 취득·처분·소각 결정) |
| 연간 소각 비율 | DART 공시 (자기주식 소각 결정) ÷ 총 발행주식수 |
| 자사주 보유 비율 | DART 분기/사업보고서 (자기주식 현황) |
| 미래 성장 잠재력 | 수동 (KIS `finance_growth_ratio`, `estimate_perform` 보조) |
| 기업 경영 | 수동 (KIS `finance_profit_ratio` ROE 보조) |
| 세계적 브랜드 보유 | 수동 (Interbrand 등 외부 참고) |
