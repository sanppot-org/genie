이 프로젝트의 목표, 주요 기능 리스트

## 🎯 프로젝트 정의
- **프로젝트 명:** Genie
- **한 줄 정의:** 데이터 기반의 주식 자동매매, 백테스트 및 금융 자산 통합 관리 시스템.
- **핵심 목표:** 개인의 자산을 안전하고 효과적으로 증식시키기 위해 감정을 배제한 알고리즘 매매 및 금융 데이터 분석 환경 구축.

## 🚀 주요 기능 (Roadmap)
1. **주식 자동매매 (Trading):** 국내/해외 주식 API 연동을 통한 실시간 매수/매도 집행.
2. **백테스트 엔진 (Backtesting):** 과거 데이터를 활용한 전략 검증 및 성과 분석(MDD, 샤프 지수 등).
3. **금융 확장성 (Future):** 부동산 데이터 분석, 가상자산 연동, 포트폴리오 최적화 기능 추가 예정.
4. **대시보드:** 자산 현황 및 매매 히스토리를 시각화하여 한눈에 파악.

## 👤 대상 사용자 및 환경
- **사용자:** 알고리즘 매매를 통해 자산 증식을 꾀하는 1인 개발자(본인).
- **중요 가치:** **안정성 > 수익률.** 시스템 오류로 인한 자산 손실 방지가 최우선.

## 💡 금융 도메인 및 비즈니스 규칙
- **Data Integrity:** 백테스트 시 편향(Look-ahead bias)이 발생하지 않도록 데이터 처리에 엄격해야 한다.
- **Security:** API Key 및 계좌 비밀번호는 절대 코드 내에 하드코딩하지 않으며, 환경 변수(`.env`)를 통해서만 관리한다.

## 📍 현재 진행 단계
- 현재 기본 아키텍처 및 백테스트 엔진의 기초를 설계 중이며, AI 에이전트를 통한 전략 고도화 단계를 준비하고 있음.
